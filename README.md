# React SPA Template

Comes with...

- React (TS + SWC)
- React Router
- Zustand
- Tailwind CSS
- DaisyUI
- Lucide React
- PWA
- Vite, ESLint, Prettier

## Usage

- npm

```
npx degit krishna-santosh/react-spa-template
```

- pnpm

```
pnpx degit krishna-santosh/react-spa-template
```

- bun

```
bunx degit krishna-santosh/react-spa-template
```
